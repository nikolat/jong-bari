■概要

  これは「伺か」で動作するゴーストです。
  麻雀をするためのサーバーゴーストです。
  他にクライアントゴーストを3体召喚してお楽しみください。

□現状の説明
- 七対子の形式しか対応していない
  - 七対子以外でも和了れるが役と点数計算は未対応
  - 鳴き未実装
- balloonにすべてを出力している
  - 選択肢による操作のみ受け付けている
- クライアントゴーストとの通信はそこそこ上手く動いている
  - 捨て牌の問い合わせに空文字を返すorタイムアウトの場合は
    サーバー側で「お任せ」の処理を行う
    ※ひたすら七対子を目指す単純処理となっている
